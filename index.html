Sobre Mim

Olá, meu nome é Juliamel! Sou estudante de tecnologia e gosto muito de criar sites e interfaces para a internet. No meu dia a dia, estudo como estruturar páginas com HTML, deixar o visual bonito com CSS e criar interações usando JavaScript.
